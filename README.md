# CodeAlpha Python Internship Project

#Task 1: Hangman game
A text based hangman game built in python. The computer picks a random word and the player guesses letters one at a time.
The player has 6 attemps before losing.

##Features
- Random word selection
- Tracks correct and wrong guesses
- Displays the remaining attempts
- Win and lose message

## How to RUN
1. Make sure python is installed.
2. Run 'hangman.py'

## Concepts used
- random library
- while loops
- lists and strings
- if/else statements
